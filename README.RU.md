# Emotion Book

Плагин для [Obsidian](https://obsidian.md), который позволяет отслеживать своё эмоциональное состояние прямо внутри ежедневных заметок.

## Как это работает

Вставь блок в любую заметку:

````
```emotion-book
```
````

Появится форма с выбором эмоции и текстовым полем. Заполни и нажми **Добавить запись** — запись сохранится с временной меткой. За день можно добавлять сколько угодно записей. Все прошлые записи прячутся под коллапсом, чтобы заметка оставалась чистой.

После каждой записи плагин автоматически считает среднюю эмоцию за день и записывает её в свойство `emotion` в метаданных заметки. Это позволяет легко делать запросы через Dataview по всему хранилищу.

## Возможности

- Быстрое добавление эмоции с цветовой маркировкой
- Несколько записей в день с временными метками
- Редактирование и удаление любой записи прямо в блоке
- Автоматический расчёт средней эмоции за день в свойство `emotion`
- Полностью настраиваемый список эмоций — названия, цвета и числовые веса
- Color picker для каждой эмоции в настройках

## Установка

1. Скачай `main.js` и `manifest.json` из последнего релиза
2. Создай папку `.obsidian/plugins/emotion-book/` в своём хранилище
3. Положи оба файла внутрь
4. Открой Obsidian → Настройки → Сторонние плагины → включи **Emotion Book**

## Настройки

Настройки → Emotion Book. Там можно добавлять, переименовывать и удалять эмоции, выбирать цвет для каждой и задавать числовой вес для расчёта среднего.
