# Emotion Book

**Emotion Book** — это плагин для [Obsidian](https://obsidian.md), который добавляет структуру в ваши ежедневные заметки. Фиксируйте своё состояние в течение дня, отслеживайте динамику и храните записи прямо в отдельных Markdown-файлах.

![alt text](<image/Cap 2026-07-16 at 19.31.58.gif>)

## Основные возможности

* **Быстрый ввод через диалог:** Выберите эмоцию из списка, нажмите **Добавить запись** и опишите момент в всплывающем окне. Хронологическая лента дня показывается ниже, прямо в ежедневной заметке.
* **Гибкая настройка эмоций:** Создавайте собственные эмоции, настраивайте для них цвета и веса (0–10).
* **Умное определение даты заметки:** Запись сохраняется в папку того дня, к которому относится заметка. День определяется по frontmatter-полю (настраивается, по умолчанию `created`), затем по имени файла, затем по дате создания файла.
* **Чистый Markdown:** Все новые записи сохраняются как отдельные `.md` файлы с понятным YAML Frontmatter (теги, дата, эмоция), что позволяет легко анализировать их через Dataview или другие плагины.
* **Поддержка старого формата:** Совместимость с блоками `emotion-book`, содержащими JSON-данные внутри кода.
* **Двуязычный интерфейс:** Полная поддержка русского и английского языков.

## Установка

### Ручная установка

1. Скачайте файлы `main.js`, `styles.css` и `manifest.json` из последнего релиза
2. Перейдите в папку вашего хранилища Obsidian по пути: `.obsidian/plugins/`
3. Создайте папку emotion-book и скопируйте в неё скачанные файлы.  

### Автоматическая установка

1. Откройте магазин плагинов Obsidian
2. Найдите плагин 
3. Установите

## Как использовать

Добавление блока на страницу

Чтобы встроить дневник эмоций в ежедневную (или любую другую) заметку, добавьте следующий блок кода:

````
```emotion-book
```
````

![alt text](<image/Cap 2026-07-16 at 19.22.03.gif>)

Дальше всё просто: выберите эмоцию в выпадающем списке, нажмите **Добавить запись**, напишите пару слов в диалоговом окне и сохраните. Запись появится в ленте дня ниже.

## Настройки

В настройках плагина (Настройки → Emotion Book) вам доступны:

1. Язык (Language): Переключение между русским и английским языками интерфейса.
2. Папка: Путь к папке, в которой будут автоматически создаваться файлы записей.
3. Поле даты во frontmatter: Имя frontmatter-поля заметки, из которого берётся дата записи (по умолчанию `created`). Если поля нет — дата берётся из имени файла, затем из даты создания файла.
4. Список эмоций: Управление названием, цветом и числовым значением (0–10) каждой эмоции.
5. Сброс: Быстрый возврат к стандартному набору эмоций.
